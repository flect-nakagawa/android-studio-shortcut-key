# Android Studio Shortcut Key

| 概要            | キー           | 備考           |
| ------------- |-------------|-------------|
| 大、小文字変換 | Cmd + Shift + u |-| 
| 自動フォーマット | Cmd + Opt + l |-| 
| 自動インポート | Ctr + Opt + o |-| 
| 単語選択 | Opt + 上矢印 |-| 
| 行削除 | Cmd + Del |-| 
| 呼び出し元 | Cmd + Alt + h |-| 
| 階層 | Ctr + h |-| 
| 参照先 | Alt + F7 |-| 
| メンバ変数とメンバメソッドの一覧 | Cmd + F12 |-| 
| メソッド、フィールドに移動 | Opt + Cmd + o |-| 
| メソッドが期待している引数を表示 | Ctr + p |-| 
| ファイルで使用している変数のハイライト | Ctr + Shift + F7 |-| 
| クイック定義確認 | Opt + Space |-|
